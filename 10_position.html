        Total 5 Position values
            -> relative
            -> absolute
            -> fixed
            -> sticky
            -> static

        i_ absolute:
            ->  When you assign abosolute the element is detached from document and can be
                arranged anywhere on page using top, bottom, left and right postions.
    
        ii_ relative:
            ->  When you assing relative the element is set relative to its parent element,
                if the parent element not found by default it relative to html tag.    
        iii_ static:
            ->  Static postions element does not have access to left, right, top or bottom
                nor z-index properties.
        iv_ fixed:
            ->  When you assign fixed to element it removed from document and it always
                relative to html tag.
        v_ sticky:
            ->  When you assign sticky to element it will stick from the point it will 
                reach to any of top, bottom, left or right proprty.
