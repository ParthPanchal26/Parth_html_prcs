            The HTML audio element is used to play an audio file on a web page.
            The controls attribute adds audio controls, like play, pause, and volume.
            The autoplay attribute start a audio automatically.
            The muted attribute will mute your audio sound.
            The source element allows you to specify alternative audio files in src attribute
            which the browser may choose from. The browser will use the first recognized format.
            The text written in between the audio and audio tags will display only if browser do
            not support the audio element.
